# FILESERVER-GO

## For development

* Install development tools.
  * Go version must be >= 1.11
* Prepare source:

```bash
make gomod;
```

* Use `make` or `make help` to see available options.

```text
make <option>
- all | help: Show help
- gomod: Prepare Go modules
- build: Build product
- run: Execute build then run immediately
- clean: Clean all outputs
```
